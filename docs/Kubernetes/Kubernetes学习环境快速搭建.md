根据你的学习目标，我推荐以下方案（从简单到复杂）：

特点:

⚡ 5-10分钟快速搭建多节点集群

💾 资源占用小（4-8GB内存即可）

🥈 方案二：Minikube ← 平衡选择
特点:

🔧 支持多种驱动（Docker/VirtualBox/Hyper-V）

适合场景:

系统学习K8s
需要测试不同网络插件

💪 接近真实虚拟机环境

适合场景:

想要虚拟机但更简单
资源有限（8-12GB内存）
学习自动化部署
